Readme for MyVote.UI.Web.NetCore
**********************************************
-This is the current application used during Modern Apps LIVE! Presentations
-It is a hybrid bootstrapped Angular1 + Angular2 application, using TypeScript, Typing, NPM, Gulp, and ASP.NET Core
-The API compliment to this web app is 'MyVote.Services.AppServer.Core' which is the current ASP.NET Core targeting .NET Core implementation
